Python Warning Test¶
print "Hello world"
assert 5 / 2 == 2

In Python 3, you need to add parentheses around "Hello world", and the division call will return 2.5 instead of 2.

Lisp¶

(defparameter *small* 1)
(defparameter *big* 100)

(defun guess-my-number ()
     (ash (+ *small* *big*) -1))

(defun smaller ()
     (setf *big* (1- (guess-my-number)))
     (guess-my-number))

(defun bigger ()
     (setf *small* (1+ (guess-my-number)))
     (guess-my-number))

(defun start-over ()
   (defparameter *small* 1)
   (defparameter *big* 100)
   (guess-my-number))
